VR Beat Saber map maker that converts your physical movements into a Beat Saber map

Project and Website very much WIP

Download User Guide

Launcher

Configure your map, pick the song and more using our cross-platform launcher. It also handles all the machine learning that is needed for generating the blocks

Supports your VR headset

Any VR headset that is supported by SteamVR is supported by BeatSketch. Simply connect it as you would to play BeatSaber

Easy to use

A simple user interface to set up your map and our user guide help you get your map right - the first time
