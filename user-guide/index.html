Installing BeatSketch

  1. Download for your platform
  2. Unzip / Untar, run BeatSketchLauncher file

Preparing the Audio

  1. Determine bpm (e.g. with bpm from bpm-tools on Linux)
  2. Open Audio editor (such as Audacity), set the BPM you just got, then open the audio file
  3. Line up bars with the song
  4. Make one or two bars of space at the start of the song (minimum 2s, max 8s, for long intros, maybe shorten the intro)
  5. BPM should now line up perfectly with the song
  6. Export the song as Ogg Vorbis Files, whole project
